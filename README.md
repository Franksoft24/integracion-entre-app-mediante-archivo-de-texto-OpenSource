#Hola!!!

Este programa es un ejemplo de integración, mediante archivo de texto entre 2 aplicaciones desarrolladas
bajo tecnología open source, con dos lenguajes distintos (PHP y JAVA) y bajo dos entornos diferente una es
console aplication y la otra es una web aplication.

Para el uso de esta app debe tener en cuenta lo siguiente:

 1-	El archivo que esta genera se guardara en un directorio dentro de la aplicacion php.

 2-	Debe de ejecutar los script de la base de datos, ambos se encuentran en la carpeta: Script bd.
	Para la prueba se utilizo mariaDB pero puede utilizar el que más le convenga.

 3-	En el código deberá de cambiar el connection string, al que se adapte a su pc.

 4-	Es odvio, pero debera de tener un servidor que corra código php para la ejecución de la app,
	se aconseja XAMPP o wamp, pero puede utilizar el que más le convenga.

Eso es todo!!!. #ES FACIL!!!!! :D
